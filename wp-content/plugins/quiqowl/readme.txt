== Description ==

== Installation ==

= Minimum Requirements =

* WordPress 6.0 or newer
* WooCommerce 7.6 or newer
* PHP 7.3 or newer (recommended)

= Installation =

Install and activate the [WooCommerce](https://woocommerce.com/) and [QuiqOwl](https://untapwp.com/quiqowl/) plugins. Once both are activated, you're all set to go.

= Direct from the Dashboard =
1. Go to WP Admin Dashboard > Plugins > Add New
2. Search for "QuiqOwl"
3. Click Install Now, and then activate the plugin.
